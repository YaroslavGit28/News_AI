# Roadmap

## Q1
- Подключить PostgreSQL и миграции Alembic.
- Добавить OAuth-вход и хранение предпочтений пользователя.
- Нормализовать ingestion через message broker (RabbitMQ/Kafka).

## Q2
- Интегрировать LLM для продвинутых резюме и пояснений.
- Встроить семантический поиск (OpenSearch/pgvector).
- Реализовать уведомления (email, Telegram-бот).

## Q3
- Личный ассистент в чат-формате (RAG поверх архива статей).
- A/B-тесты рекомендаций и адаптивные дэшборды.
- Облачный деплой + IaC (Terraform, Pulumi).
