# Persona News — Персональный агрегатор новостей

**Проблема:** информационная перегрузка и отсутствие структуры в лентах.

**Цель:** собрать новости из разных источников, отфильтровать шум и помочь пользователю быстро понять, что важно именно для него.

## Что входит

- `backend/` — FastAPI + Celery за сбор, нормализацию и персонализацию контента.
- `frontend/` — Next.js-дэшборд с темами, фильтрами и объяснениями.
- `docs/` — архитектура, модель данных, roadmap.
- `infrastructure/` — docker-compose и шаблоны конфигураций.

## Новые функции

✅ **Русские источники** — РИА Новости, ТАСС, Интерфакс, Lenta.ru, Meduza, RBC, Ведомости, Коммерсант, Хабр, VC.ru и др.  
✅ **Международные источники** — BBC, Reuters, The Guardian, TechCrunch, The Verge и др.  
✅ **15 категорий тем** — Технологии, Экономика, Геополитика, Общество, Культура, Наука, Спорт, Медиа, Экология, Транспорт, Недвижимость, Образование, Здоровье, Развлечения, Кибербезопасность  
✅ **AI ассистент** — разговорчивый помощник для анализа новостей, поиска по темам и обсуждения трендов  
✅ **Автоматический перевод** — новости автоматически переводятся на русский язык  
✅ **Фильтры и персонализация** — тематические теги, рекомендации по лайкам/дизлайкам  
✅ **Индикаторы свежести** — цветные метки и время публикации  
✅ **Обратная связь** — лайк/дизлайк/скрыть + возможность вернуть статью в течение часа

## Быстрый старт (локально)

### Вариант 1: Одна команда (рекомендуется)

**Windows:** просто запустите `start-dev.bat` (двойной клик или из терминала):
```bash
start-dev.bat
```

**Или через npm (после установки зависимостей):**
```bash
npm install  # установить concurrently
npm run dev  # запустить оба сервера одновременно
```

> Скрипт `start-dev.bat` автоматически создаёт `backend\.venv`, устанавливает python-зависимости и выполняет `npm install` во `frontend/`, если это ещё не сделано. После старта он проверяет `/health`, чтобы убедиться, что API отвечает.

### Вариант 2: Вручную (два терминала)

```bash
# Backend
cd backend
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload

# Frontend (в другом терминале)
cd ../frontend
npm install
npm run dev
```

Откройте http://localhost:3000 в браузере.

## API Endpoints

- `GET /health` — статус сервиса (время старта, количество статей)
- `GET /feed?user_id=demo&limit=25` — персональная лента + список скрытых статей
- `GET /sources` — подключённые RSS-источники
- `POST /feedback?article_id=1&action=like|dislike|hide|undo_hide` — обратная связь
- `POST /chat` — общение с AI ассистентом

## Технологии

- **Backend**: FastAPI, Python 3.12, Uvicorn
- **Frontend**: Next.js 14, React 18, TypeScript
- **AI**: Локальная логика анализа и классификации новостей
- **Источники**: RSS фиды из 20+ источников
