# Frontend (Next.js)

Приложение Next.js отображает персональную ленту из backend API:

- фильтры по темам и источникам;
- карточки с изображениями, бейджами тем и индикаторами свежести;
- лайк/дизлайк/скрыть + возможность вернуть скрытую статью в течение часа;
- шапка с поиском и быстрым обновлением.

## Запуск
```bash
npm install
npm run dev
```

По умолчанию ожидает backend на `http://localhost:8000`. При необходимости обновите `frontend/lib/api.ts`.

## Что нового в UI
- Поиск по заголовку, сущностям, источнику и темам прямо в шапке
- Панель управления лентой с сортировкой (рекомендации/свежесть/источник) и переключением Grid/List
- Быстрая полка «Избранное» и звездочка на карточке для сохранения статьи
- Улучшенные карточки: индикаторы тональности, юзабилити на планшете/мобиле, подсветка свежести

## Советы по дев-среде
- Если PowerShell блокирует `npm`, запустите команды из `cmd.exe` или разрешите выполнение скриптов (`Set-ExecutionPolicy -Scope CurrentUser RemoteSigned`)
- Для линтинга используется `next lint`. При первом запуске выберите опцию Strict, чтобы сохранить `.eslintrc` и отключить повторные вопросы
